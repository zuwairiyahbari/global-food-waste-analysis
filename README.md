# Global Food Waste Analysis

## Overview

This project digs into food waste data across countries, using regression models to figure out what actually drives waste levels and whether there's a clear split between developed and developing nations.

## Research Questions

- Can we predict total food waste just from population size and how much households toss out?
- Do rich and poor countries show fundamentally different waste habits?

## Methods

- Cleaned up messy data and got it model-ready
- Ran multiple linear regression models
- Tested different variable combinations with best subset selection
- Used 10-fold cross-validation to keep overfitting in check
- Evaluated models with RMSE, MAE, and MSE

## Tools Used

- R (plus the usual tidyverse suspects)
- Regression modeling
- Cross-validation
- Plots and charts to see what's going on

## Key Findings

- Economic loss turned out to be the biggest driver of total waste—more than population or household habits.
- Household waste and population still added something to the model, just not as much.
- Whether a country was "developed" or "developing" didn't really move the needle on its own.
